📊 Blinkit Sales Dashboard | Microsoft Excel
Project Overview

Developed an interactive Blinkit Sales Dashboard in Microsoft Excel to analyze sales performance, customer ratings, outlet distribution, and product categories. The dashboard provides business insights through KPIs, Pivot Tables, Pivot Charts, Slicers, and data visualization, enabling users to monitor sales trends and make data-driven decisions.

Features
Interactive dashboard with dynamic slicers
KPI Cards displaying:
Total Sales: $0.24M
Average Sales: $141
Number of Items: 8,523
Average Rating: 4.0
Sales trend analysis by outlet establishment year
Sales distribution by outlet size (High, Medium, Small)
Sales analysis by outlet location (Tier 1, Tier 2, Tier 3)
Item type-wise sales comparison
Fat content analysis (Low Fat vs Regular)
Outlet type performance analysis
Interactive filtering using slicers
